# On Stuby React.js

Blog Web Application for learning purposes using React.js.

## Development Environment
- SPA : react-scripts
  - Application Server
  - port
    - 8031
- API : json_server
  - API Mock Server
  - port
    - 8032

## build
```shell
docker compose build
```

## up
```shell
docker compose up
```

## down
```
docker compose down
```
